\# pytest-api-automation



基于 Pytest 的轻量级接口自动化框架。



\#核心功能

&nbsp;    并发执行\*\*：集成 `pytest-xdist`，支持多线程运行。

&nbsp;    环境隔离\*\*：预设 `venv` 规范与 `requirements.txt`。

&nbsp;    参数化配置\*\*：通过 `pytest.ini` 实现 CLI 参数固化。

&nbsp;    结果交付\*\*：集成 `pytest-html` 生成单文件自动化报告。



\#开始



\# 1. 激活环境

venv\\Scripts\\activate



\# 2. 安装依赖

pip install -r requirements.txt



\# 3. 运行测试

pytest
